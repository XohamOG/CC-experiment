# Project

This workspace contains two folders:

- `Frontend/` – a Vite + React scaffold
- `Backend/` – an Express scaffold

To run locally (PowerShell):

Frontend:

```powershell
cd "D:/CC experiment/Frontend"
npm install
npm run dev
```

Backend:

```powershell
cd "D:/CC experiment/Backend"
npm install
npm run dev
```

Open http://localhost:5173 for the frontend and http://localhost:3000 for the backend.
